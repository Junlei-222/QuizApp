# QuizApp for SIT305

A multiple-choice quiz Android app with real-time progress tracking, visual feedback (green/red highlights), and session persistence.

## Features
- Submit answer: correct turns green, wrong shows red on selected and green on correct.
- Progress bar updates after each question.
- Final score activity with "Take New Quiz" (retains user name) and "Finish" button.
- Built with Java, no external libraries.

## How to Run
1. Clone this repository.
2. Open in Android Studio.
3. Build and run on an emulator or Android device (API 21+).

## Subtask Completion (SIT305)
- Subtask 1: Answer selection & visual feedback 
- Subtask 2: Real-time progress tracking 
- Subtask 3: Final score & session persistence  
- Subtask 4: Research report – see submission PDF
